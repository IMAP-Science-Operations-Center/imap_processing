Project Management
==================

The project management pages describe how the IMAP SDC project will be managed throughout its lifecycle. The project stages covered in this section include *Requirements and Architecture* through *Integration, Test, and Verification* as shown in the diagram below.

.. image:: https://web.archive.org/web/20090705103034im_/http://www.itsdocs.fhwa.dot.gov/jpodocs/repts_te/14158_files/image015.gif
   :alt: systems engineering process diagram
   :width: 650
   :height: 400


The following pages are meant to answer the following questions:

* How are SDC requirements being tracked, implemented, and verified?
* How are SDC issues being managed in Github throughout the project's lifecycle?
* What process does the SDC follow for its Software Integration Tests?
